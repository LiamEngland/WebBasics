Basic Dropdown

Hover

This is a simple dropdown.

This works by having any element to open the dropdown, in this case being span being held in a div. This will also contain another div where the content will be held.

The parent div has a position of relative, so the dropdown content can be placed absolutely when it's visible. By default this content's display is set to none, and only when the parent element is hovered, will the content div have the display set to block where the content can be shown.
