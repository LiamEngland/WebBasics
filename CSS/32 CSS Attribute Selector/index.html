CSS Attribute Selectors

It's possible to style HTML elements that have specific attributes or attribute values.

CSS [attribute] Selector:

The [attribute] selector is used to select elements with a specific attribute.

Below's an example which will select all Anchor tags with a target attribute.

I am a link with a target attribute I am a link!

CSS [attribute='value'] Selector:

The [attribute='value'] selector is used to elements with a specified attribute and value.

Below's an example that selects all elements with a target='blank' attribute.

I am another link with a target attribute I am just a link!

CSS [attribute~='value'] Selector:

The [attribute~='value'] selectr is used to select elements with an attribute value containing a specific word.

Below's an example that selects all elements with a title values that contain the word "testing".

Title: "testing"

Title: "test"

Title: "testing selectors"

CSS [attribute|='value'] Selector:

The [attribute|='value'] selector is used to select elements with a specified attribute starting with the specified value.

Below's example selects all elements with a class attribute value that begins with 'top'.

One thing to note is the value has to be a whole word, whether that be alone, or followed by a hyphen.

I have "top" in my class.

I don't.

CSS [attribute^='value'] Selector:

The [attribute^='value'] selector is used to select elements with values starting with the value specified. Using this method the value in question doesn't need to be the whole word.

The below's an example of which selects all attribute values starting with "top".

I have a class starting with 'top'.

I have a class that contains 'top'.

My class value is just 'top'.

CSS [attribute$='value'] Selector:

The [attribute$='value'] selector is used to select elements which values end with a specified value.

The below's an example of selecting all elements with a class attribute ending with "test". One thing to note is the value doesn't need to be one word.

My class ends with test.

My class doesn't end with test.

My class is test.

CSS [attribute*='value'] Selector:

The [attribute*='value'] selector is used to select elements whose attribute value contains a specific value.

Below's an example of selecting elements with a class containing a value that contains "te". One thing to note, the value doesn't need to be a whole word.

Test One.

Test Two.

Test Three.

Test Four.
