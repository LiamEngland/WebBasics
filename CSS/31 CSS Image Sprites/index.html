CSS Image Sprites

An image sprite is a collection of images put into a single image.

Web pages with many images can take a long time to load and generates multiple server requests.

Using image sprites will reduce the number of server requests and save bandwidth.

Simple Example:

Rather than using three separate images, a single image can be used.

With CSS, only a part of the image is needed to be shown.

This is done by setting either an ID or a Class with a width, height and background, set to the image in question. The background is the important part, as it allows a background image to be defined, and the position of which part of the image you want showing.

Image Sprite Navigation List:

By using the same image, a navigation list can be created.

By setting a new image, this has darkened icons, which can be used for hover effects. So rather than loading 6 individual images, there's just one, which speeds up the process.
